## What You Need to Provide

### 1. Trained Model Pickle
- Generated by: `run_ours_train_lr_paper_custom.py`
- Must contain: `model`, `scaler`, `flip_mask`, `final_keys`
- Example: `trained_model_gpt2.pkl`

### 2. Test Dataset (JSONL)
- Format: `{"source_file": "BookName.txt", "text": "..."}`
- At least 10 texts (more = better results)
- Example path: `d:/datasets/test_books.jsonl`

### 3. Configuration File
- Already exists: `conf/config.yaml`
- Make sure `base_model` matches your training setup

## Usage Command


## Pipeline Architecture

```
Input JSONL (with book names)
    ↓
[1. Load] → texts + book identifiers
    ↓
[2. Extract Features] → token probabilities
    ↓
[3. Generate Repeated] → 5x variant features
    ↓
[4. Compute Signals] → 12+ statistical features
    ↓
[5. Load Model] → trained classifier
    ↓
[6. Reconstruct] → feature matrix in training order
    ↓
[7. Apply Mask] → sign-alignment (flip_mask)
    ↓
[8. Infer] → membership probabilities [0, 1]
    ↓
[9. Aggregate] → group by source book
    ↓
[10. Report] → TPR per book + overall stats
    ↓
Output: Console report + CSV
```

## Technical Highlights

- **Feature Extraction**: Uses same `LanguageModel` class as training
- **Feature Reconstruction**: Exact order matching (uses `final_keys`)
- **Sign Alignment**: Applies `flip_mask` for discriminative direction
- **Statistical Tests**: Slope, entropy, LZ complexity, token diversity, etc.
- **Aggregation**: Automatic grouping by `source_file`
- **Error Handling**: Robust to short texts, malformed JSON, missing fields

## FAQ

**Q: How long does it take?**  
A: 1-2 min for 100 texts on GPU, 5-10 min for 500 texts

**Q: Do I need GPU?**  
A: No, but GPU is 10x faster. CPU works fine for small datasets.

**Q: What's a good contamination rate?**  
A: 0-5% is normal, 5-15% is moderate, 15%+ indicates significant overlap

**Q: Can I change the detection threshold?**  
A: Yes! Use `--threshold 0.7` for stricter detection

**Q: What if texts are too short?**  
A: Texts with < 3 words are automatically skipped (normal)

**Q: How is contamination rate calculated?**  
A: TPR = (number of flagged texts / total texts) × 100%
